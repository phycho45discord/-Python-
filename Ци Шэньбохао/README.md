# external_sort

对一个超大的 int32 二进制文件做升序排序。文件太大装不进内存时用这个。

## 怎么用

```bash
python3 external_sort.py 1.bin 1000000
make test            # 跑测试
make clean
```

## 命令

```
python3 external_sort.py <文件名> <一次能装入的整数个数>
```

- 文件名：要排序的二进制文件，里面是一个接一个的 int32（没有头部）
- 一次能装入的整数个数：每块最多放多少个 int32

输出文件叫 `<文件名>.sorted`，放在原文件同一个目录里。

## 大概原理

1. 把大文件切成小块
2. 多进程并行：每块自己读进内存、`list.sort()`、写到临时文件
3. 用一个最小堆把所有临时文件按 K 路归并，合成一个排好的大文件
4. 删掉临时文件

## 时间复杂度

- 第一步并行：O(n · log(k))，k 是块大小
- 第二步归并：O(n · log(n/k))
- 加起来：O(n · log n)
